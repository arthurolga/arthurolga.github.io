Projetos Eletrônica Inspermon


Inspermon



Descrição

O projeto visa desenvolver um RPG em texto em Python. O tema do jogo era algo semelhante à Pokémon, porém com uma tematica ligada ao Insper. Dessa forma o jogo deveria conter uma database de monstros, evolucoes, sistema de batalha e de saves em JSON.

Competências
  • code Programação
  • create Design
class
Metodologia

O código foi feito de forma que rodasse inteiramente no terminal do Python. Dessa forma o jogo não tinha apelos visuais que

build
Técnicas Aplicadas

A dupla utilizou de conhecimentos básicos em Python, como loops simples e Fors, além disso salvamento de arquivos em JSON, de forma que os checkpoints eram salvos dessa forma.

grade
Recepção

O projeto cumpriu o esperado e teve adendos que acrescentaram uma nota maior, como sistema de equilibrio de monstros, além de um sistema randomico de criação instanea de monstros, com base no próprio Pokémon.
