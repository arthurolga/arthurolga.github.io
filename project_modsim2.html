Resfriamento de um Processador



Descrição

O projeto desenvolveu uma modelagem física do resfriamento de um processador Intel i7 com um cooler Hypercooler Xt3 com base em equações diferenciais térmicas.

Competências
  • code Programação
class
Metodologia

O código foi feito em iPython Notebooks e os dados foram coletados de fontes dos próprios fabricantes. As questões físicas forma pesquisadas em livros e assistidas pelo professor.

build
Técnicas Aplicadas

Foi utilizado um conhecimento sobre equações diferenciais, de física e de Python

grade
Recepção

O projeto mostrou-se sensato e com medidas reais, sendo validado por comparação à gráficos reais de resfriamento de computadores.
