Estação Metereológica



Descrição

Projeto com o objetivo de desenvolver uma estação metereológica capaz de medir temperatura, umidade e altitude. Para isso desenvolvemos um shield de arduino com 3 sensores que permitiam fazer tais medições.

Competências
  • code Programação
  • create Design
Link Relatório
class
Metodologia

O projeto se deu pela construção de um circuito em protoboard, posteriormente impresso em uma placa de cobre e os sensores adicionados.

build
Técnicas Aplicadas

O grupo utilizou teoria de elétrica, haiblidades de solda e prototipagem. Além disso, foram conduzidos experimentos de calibragem.

grade
Recepção

O projeto foi muito bem recebido pelos professores, de forma que atingiu nota máxima.
